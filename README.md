# CahasEDI Dashboard

This web app is the frontend for [CahasEDI-Core](https://github.com/greasysock/CahasEDI-Core)

The feature list is small but growing.

### Features

* Administrative control of users. Closed registration.
* List all messages
* List messages by type
* Invoice view
